# Regra Universal: NUNCA Abrir Navegador

**PROIBIDO** usar `browser_subagent`, `open_browser_url`, ou qualquer ferramenta que abra/acione navegador.
**PROIBIDO** gerar comandos `open`, `xdg-open`, `start` ou qualquer equivalente para abrir URLs.

O usuário verifica visualmente por conta própria.
O agente deve fazer commit e informar o que mudou — NUNCA abrir browser.
